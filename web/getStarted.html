This application uses the myanimelist.net API, so you will have to follow a few steps before you can use the app.    info


Steps to do that:

  1. Create an account on myanimelist.net
  2. Set up a Client API ID myanimelist.net/apiconfig    (More details click here )

    Go to profile settings and click on "API". After this, you will have to set up a new client API ID.


    After that, fill the fields with your information and click on "Submit".


    Next, click on "Edit" button.


    Finally, copy your Client ID.


  3. Put the Client ID in the text box below
